# Build Minimalist Portfolio

## Goal
Build a premium, high-performance developer portfolio with Next.js, Framer Motion, and Tailwind CSS using a minimally disruptive aesthetic, advanced scroll triggers, and sharp geometry.

## Tasks
- [x] Task 1: Scaffolding → Run `npx create-next-app` initialization.
- [x] Task 2: Install dependencies → Run `npm install framer-motion lucide-react clsx tailwind-merge`.
- [x] Task 3: Set up Data Structure → Create `src/data/portfolio.ts`.
- [x] Task 4: Base Layout & Typography → Configure fonts (Syne & Inter) and `globals.css` with strict sharp geometry and Acid Green accents.
- [x] Task 5: Component - Custom Cursor → Build a custom cursor with Framer Motion.
- [x] Task 6: Section - Massive Typographic Hero → Build dynamic enter animation avoiding the "safe split" layout.
- [x] Task 7: Section - Vertical Narrative About → Build the about section with stagger scroll reveals.
- [x] Task 8: Section - Fragmented Projects Layout → Build project cards with asymmetric grid and extreme hover physics.
- [x] Task 9: Section - Minimal Skills & Contact → Build final sections with sheer minimal typography.
- [x] Task 10: Assembly & Scroll Orchestration → Combine sections into `page.tsx`.

## Done When
- [x] Site runs cleanly with `npm run dev`.
- [x] Dark theme is perfectly implemented with zero purple/blue defaults.
- [x] Advanced scroll animations fire correctly without layout jank.
- [x] No generic template structures.
